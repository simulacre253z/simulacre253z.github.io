속성(Attribute)과 값(Value)

태그(요소)의 기능을 확장하기 위해 ‘속성’을 사용할 수 있음

< TAG 속성 = "값" > < /TAG >
요소는 여러 속성을 가짐
태그 이름 : p / 속성명 : class / 속성값 : value / 내용 : 단락

HTML 일반 속성 참조


0. Form
  • 사용자가 웹 사이트로 정보를 보낼 수 있는 요소 form
  • 폼과 관련된 대부분의 작업들은 정보를 저장, 검색, 수정
  • 이런 작업은 모두 DB를 기반

method : 사용자가 입력한 내용들을 어떤 방식으로 서버 쪽에 넘겨줄지 지정

  1. get : 사용자가 입력한 내용이 주소 표시줄에 그대로 나타남
  2. post : 사용자의 입력을 standard input로 넘겨 입력 길이 제한이 없고 입력 내용도 드러나지 않음

name : form의 이름을 지정. 한 문서에 form 태그가 여러 개 있을 경우 구분하기 위해 사용

action : form 태그 안의 내용들을 처리해줄 서버 상의 프로그램을 지정

target : action 태그에서 지정한 스크립트 파일을 열 위치를 지정

  • 다음과 같은 속성값을 사용할 수 있다.
    1. _self : target 속성의 기본값으로, 현재 창에서 열림
    2. _blank : 새 창이나 새 탭에서 열림
    3. _parent : 부모 프레임에 표시
    4. _top : 프레임을 벗어나 전체 화면에 표시
fieldset 태그 - 영역 묶기

영역 제목

fieldset 태그는 form 요소를 하나의 그룹으로 묶는 태그

legend 태그는 fieldset으로 묶은 구역에 제목을 붙여주는 태그


1. input : 입력과 관련한 태그 및 속성

a. 텍스트

  • text : 한 줄짜리 텍스트를 입력할 수 있는 텍스트 상자
  • password : 비밀번호 입력 필드
  • search : 검색 상자
  • url : URL 주소 입력 필드
  • email : e-mail 주소 입력 필드
  • tel : 전화번호 입력 필드

b. 숫자

  • number : 숫자 조절 화살표 필드
  • range : 숫자 조절 슬라이드 막대

c. 선택

  • checkbox : 주어진 항목에서 2개 이상 선택 가능한 체크박스
  • radio : 주어진 항목에서 1개만 선택 가능한 라디오 버튼
  • color : 색상표

d. 날짜

  • date : 사용자 지역 기준 날짜(년, 월, 일) 입력 필드
  • month : 사용자 지역 기준 날짜(년, 월) 입력 필드
  • week : 사용자 지역 기준 날짜(연, 주) 입력 필드
  • datetime : 국제 표준시(UTC) 로 설정된 날짜, 시간 입력 필드
  • datetime-local : 사용자 지역 기준 날짜, 시간 입력 필드
  • time : 사용자 지역 기준 시간(시, 분, 초, 분할 초) 입력 필드

e. 버튼

  • submit : 서버로 전송하는 버튼
  • reset : 리셋 버튼
  • image : submit 버튼 대신 사용할 이미지
  • button : 일반 버튼

f. 기타

  • file : 파일 첨부 버튼
  • hidden : 사용자에게는 보이지 않지만 서버로 넘겨지는 값

2. input 태그 속성들

a. autofocus - 페이지 불러오자마자 폼 요소에 마우스 커서가 표시

b. placeholder - 입력란에 어떤 걸 입력하는지 힌트를 표시

c. readonly - 내용을 볼 수만 있고 입력은 불가

d. required - 필수적으로 내용을 입력해야 submit 가능

e. min, max, step

  • 날짜 관련 또는 숫자 관련 타입인 경우에 사용
  • max, min : 해당 필드의 최대, 최소 값
  • step : 허용된 범위 내의 숫자 간격

f. size, minlength, maxlength

  • 텍스트 관련 타입인 경우에 사용
  • size : 화면에 몇 글자 보이게 할지 지정
  • maxlength, minlength : 최대 입력 가능 문자 길이를 지정

3. 그 외의 Form 요소들

a. select, optgroup

여러 옵션 중 선택할 수 있는 드롭다운 목록을 구성하는 태그

  • select 태그의 속성
    1. size : 화면에 표시될 드롭다운 메뉴의 항목 개수를 지정
    2. multiple : 브라우저 화면에 여러 개의 옵션이 함꼐 표시되면서 Ctrl 키를 누른 상태로 드롭다운 메뉴에 있는 여러 항목을 선택할 수 있음
  • option 태그의 속성
    1. value : 옵션 선택 시 서버로 넘겨질 값을 지정
    2. selected : 화면에 표시될 때 기본 선택되어 있는 옵션을 지정

b. datalist, option - 선택한 값을 입력

데이터 목록에서 선택한 값이 자동으로 연결된 텍스트 필드에 입력

  • value : 항목 선택시 서버로 넘겨질 값을 지정
  • label : 브라우저에 표시할 레이블, 따로 지정하지 않을 경우 value 값을 레이블로 사용

c. textarea - 텍스트 영역

  • cols : 텍스트 영역의 가로 너비를 문자 단위로 지정
  • rows : 텍스트 영역의 세로 길이를 줄 단위로 지정

d. button - 버튼

  • submit : form을 서버로 전송
  • reset : form에 입력한 모든 내용을 초기화
  • button : 버튼 형태만 만듬

e. output - 계산 결과

계산 결과를 브라우저에 표시해주는 태그

f. range - 해당 값 수치 드래그
